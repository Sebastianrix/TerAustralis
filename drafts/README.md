# TerAustralis — Working Drafts

This folder holds **exploratory working drafts** for the TerAustralis Incognita
Protocol. They develop ideas in depth before the stable core is consolidated into
the main litepaper (root [`README.md`](../README.md)), per the litepaper's
versioning and iteration rules.

> **Status legend**
> - ✅ **Consolidated** — the stable core is now reflected in the litepaper.
> - 🟡 **Partially consolidated** — the concept is in the litepaper; mechanical detail remains here.
> - ⚪ **Standalone** — lives here by design; not intended for the main litepaper.

## The architecture in one line

> **CrystalCore** (origin) → **Starlines** (paths) → **Navigator** (wayfinding) →
> **$TINC** (incentive) — all bound by **"one, but many"**, protected by
> **Honesty & Truth**, with **consent mechanical at every layer**.

## Working notes

- **Naming & external content.** Some concepts were sparked by cultural or copyrighted
  works (e.g. songlines, a well-known song). All drafts use **original wording**; any
  cultural or copyrighted framing is **provisional and clearance-pending** before
  external publication, consistent with the protocol's Indigenous Sovereignty &
  Consent principle and basic copyright respect.
- **Consolidation flow.** Develop here → consolidate the stable core into `main` with
  an appropriate version bump and a changelog entry → leave open mechanical questions
  in the draft until resolved.
- **Open mechanical questions** (supply figures, attestor parameters, governance
  model, Starline composition, DTN-latency handling) are tracked in the relevant
  draft's "Open questions" section.
